# AI Chatbot using Streamlit

A simple chatbot built using Python and Streamlit with API integration.

## Features
- Chat interface for user interaction
- Generates responses using external API
- Maintains chat history during session
- Basic error handling

## Tech Stack
- Python
- Streamlit
- Requests

## How to Run
1. Install dependencies:
   pip install streamlit requests

2. Run the app:
   streamlit run app.py

## About
This project was developed as part of a guided training session to understand API integration and building interactive web apps using Python.
